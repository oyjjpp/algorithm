### 题目
给定一棵二叉搜索树，请找出其中第k大的节点。

### 限制
1 ≤ k ≤ 二叉搜索树元素个数 

### 思路
二叉搜索树中序遍历是递增的顺序  

### 方案一
1.将二叉搜索树的值经过中序遍历保存到数组中  
2.保存到数组中的值是递增的，通过索引获取制定值  


### 方案二
1.中序遍历倒序方式即是递减的序列，直接递归k次即可  

#### 原理
中序遍历：左中右  
中序遍历的倒序：右中左
